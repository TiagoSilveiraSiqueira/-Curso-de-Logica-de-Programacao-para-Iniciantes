# Explicação:
Antes de escrever código de programação real, usamos o pseudocódigo para planejar nossos programas. <br /> 
O pseudocódigo é uma linguagem simples que nos ajuda a descrever os passos de um algoritmo. <br /> 
Um algoritmo é uma sequência de instruções que resolve um problema.<br /> 

# Exemplo:
Algoritmo para fazer um sanduíche:<br /> 
  Pegue duas fatias de pão.<br /> 
  Espalhe a manteiga em uma fatia.<br /> 
  Coloque o queijo e o presunto na outra fatia.<br /> 
  Junte as duas fatias e corte o sanduíche ao meio.<br />
